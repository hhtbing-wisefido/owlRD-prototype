## 🚀 部署建议

### 推荐架构
```
前端: React + Vite + TailwindCSS
后端: FastAPI + PostgreSQL
部署: Docker + Nginx
监控: Prometheus + Grafana
```

### 环境要求
```bash
# 前端
Node.js 18+
npm/yarn

# 后端
Python 3.10+
FastAPI 0.100+
PostgreSQL 14+

# 部署
Docker 20+
Nginx 1.20+
SSL证书（生产环境）
```

### 部署步骤
```bash
# 1. 克隆代码
git clone https://github.com/hhtbing-wisefido/owlRD-prototype.git

# 2. 构建前端
cd frontend
npm install
npm run build

# 3. 启动后端
cd backend
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000

# 4. 配置Nginx
# 前端: localhost:80 → frontend/dist
# 后端: localhost/api → localhost:8000
```
