## 🎯 使用指南

### 新成员入门

**步骤1**: 阅读AI规则系统
```bash
# 查看规则系统总览
cat ../../.windsurfrules/README.md
```

**步骤2**: 安装自动检查机制
```bash
# 安装Git Hook
python ../../.windsurfrules/scripts/install_git_hooks.py
```

**步骤3**: 阅读项目特定规范
- 📖 自动检查机制使用指南
- 📖 聊天记录管理规范
- 📖 项目维护清单

---

### 日常开发

**创建文件前**:
1. 遵守 [01-文件操作规则](../../.windsurfrules/01-file-operations.md)
2. 执行5步强制检查

**提交代码前**:
1. Git Hook会自动检查项目结构
2. 检查通过才能提交

**遇到问题时**:
1. 查看 [自动检查机制使用指南](./自动检查机制使用指南.md)
2. 手动运行检查: `python ../../.windsurfrules/scripts/check_project_structure.py`
