# owlRD系统演示脚本

**演示时间**: 约10分钟  
**演示人员**: 1人  
**准备时间**: 5分钟

---

## 🎬 演示前准备 (5分钟)

### 1. 启动服务

```bash
# 终端1: 启动后端
cd owlRD-prototype/backend
python -m uvicorn app.main:app --reload --host 0.0.0.0

# 终端2: 启动前端
cd owlRD-prototype/frontend
npm run dev
```

### 2. 确认服务状态

- ✅ 后端: http://localhost:8000/docs
- ✅ 前端: http://localhost:3000
- ✅ WebSocket: 连接正常

### 3. 准备演示浏览器

- 打开浏览器到 http://localhost:3000
- 清除缓存（Ctrl+F5）
- 打开开发者工具 (F12) - Network标签

## 🎯 常见问题准备

### Q1: 数据是真实的吗？
**A**: 当前使用示例数据演示。系统支持连接真实IoT设备，通过TDPv2协议接收实时数据。

### Q2: 支持多少住户？
**A**: 架构支持扩展。当前JSON存储适合原型和小规模部署。生产环境可迁移到PostgreSQL，支持数千住户。

### Q3: 告警如何通知？
**A**: 当前实现WebSocket实时推送。预留了App推送、Email、电话等多通道接口。

### Q4: 数据安全吗？
**A**: 完全匿名化设计，符合HIPAA标准。敏感信息加密存储，不暴露真实身份。

### Q5: 能定制吗？
**A**: 完全可以。模块化设计，支持定制告警规则、质量指标、UI主题等。

### Q6: 部署复杂吗？
**A**: 提供Docker容器化部署。也支持传统方式部署。有完整部署文档。
