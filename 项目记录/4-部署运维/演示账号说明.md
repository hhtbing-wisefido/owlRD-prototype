## 🚀 登录方式

### Web端登录
1. 打开: http://localhost:3000
2. 输入用户名和密码
3. 点击"登录"

### API登录测试
```bash
curl -X POST http://localhost:8000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin_user","password":"demo123"}'
```

**响应**:
```json
{
  "access_token": "eyJhbGci...",
  "token_type": "bearer",
  "expires_in": 86400,
  "user": {
    "user_id": "...",
    "username": "admin_user",
    "role": "Director"
  }
}
```
