## 🔧 故障排查

### 问题1: Git Hook不工?
**解决方案**:
```bash

chmod +x .git/hooks/pre-commit

# 或手动运?cd .\scripts
python update_project_status.py
git add .\项目记录\项目状?json
```

### 问题2: 统计数据不准?
**可能原因**:
- 新增文件扩展名未包含在扫描中
- 某些目录被错误排?
**解决方案**:
- 修改`update_project_status.py`中的扩展名列?- 检查目录排除规?
### 问题3: 服务状态检测失?
**可能原因**:
- PowerShell不可?- 端口检测超?
**解决方案**:
- 脚本会优雅处理错误，不影响更?- 服务状态仅供参?
