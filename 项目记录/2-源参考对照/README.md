```bash

# 2. 验证前端全局类型
```

### 发现问题后的流程

1. **确认问题** - 使用自动验证脚本或手动检查
2. **记录问题** - 在相应检查清单中标记
3. **修复问题** - 参考最佳实践进行修复
4. **验证修复** - 重新运行验证脚本
5. **更新清单** - 标记为已修复
