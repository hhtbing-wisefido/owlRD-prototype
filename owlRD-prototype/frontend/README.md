# owlRD Frontend

React + TypeScript前端应用

## 技术栈

- React 18
- TypeScript
- Vite
- TailwindCSS  
- React Router
- React Query
- Recharts
- Lucide Icons

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

访问: http://localhost:3000

## 构建

```bash
npm run build
```

## 功能模块

- 仪表板 - 实时数据概览
- 住户管理 - 住户信息管理
- 设备管理 - IoT设备监控
- 告警中心 - 多级告警处理
- 护理质量 - 护理质量评估

## API配置

后端API地址配置在 `.env`:
```
VITE_API_URL=http://localhost:8000
```
