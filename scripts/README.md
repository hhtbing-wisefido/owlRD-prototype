# 项目工具脚本

本目录包含项目自动化工具脚本。

---

## 📝 脚本列表

### 1. `update_project_status.py`

**功能**: 自动更新项目状态文件

**作用**:
- 扫描后端代码统计行数
- 扫描前端代码统计行数
- 统计文档行数
- 获取Git提交次数
- 检测服务运行状态
- 更新`../项目记录/项目状态.json`

**使用**:
```bash
cd scripts
python update_project_status.py
```

**自动运行**: 
- 每次`git commit`时自动运行（通过Git Hook）

---

### 2. `update_status.bat`

**功能**: Windows快捷启动脚本

**作用**:
- 双击运行`update_project_status.py`
- 显示友好的界面
- 运行后暂停查看结果

**使用**:
- 双击`update_status.bat`文件
